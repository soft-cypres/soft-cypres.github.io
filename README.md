# soft-cypres-site

Marketing and support site for **Soft-CYPRES** — a training simulator for the CYPRES 2 Changeable Mode AAD.

Deployed at: https://dhcloud.github.io/soft-cypres-site/

## Pages

- `index.html` — main marketing page
- `support.html` — FAQ and contact
- `privacy.html` — privacy policy

## Stack

Plain HTML + CSS + a few lines of vanilla JavaScript. No build step, no framework, no trackers.

## Local preview

```
python3 -m http.server 8080
```

Then open http://localhost:8080.
